# 🍲 Recipe Explorer

Recipe Explorer is an Android application built with Kotlin that allows users to browse and explore various food recipes. The app fetches recipe data from an online API and stores it locally using Room Database for faster access.

## 📱 Features
- Browse recipes by category
- View meal images and details
- Display ingredients and cooking instructions
- Open original recipe source link
- Offline storage using Room Database
- Fast API fetching using Retrofit

## 🛠 Technologies Used
- Kotlin
- Retrofit (API Integration)
- Room Database (Local Storage)
- Glide (Image Loading)
- Coroutines

## 📂 Project Structure
- **SplashScreen** – Loads categories and meals
- **HomeActivity** – Displays meal categories
- **DetailActivity** – Shows recipe details

## 🚀 How to Run
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle
4. Run the app on an emulator or physical device
